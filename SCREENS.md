# 画面構成

このアプリケーションは、シンプルな3画面構成で設計されています。

## 📱 画面一覧

### 1. トップ画面（`/`）

**ファイル**: `frontend/src/app/page.tsx`

**機能**:
- タブで画面を切り替え可能
- 動画一覧タブとアップロードタブを切り替え

**タブ構成**:
- **📹 動画一覧タブ**: アップロード済み動画の一覧表示
  - 動画が0件の場合、「まだ動画がアップロードされていません」と表示
  - 各動画カードをクリックで詳細画面へ遷移
- **⬆️ 動画をアップロードタブ**: 動画ファイルのアップロード
  - タイトル入力（任意）
  - 動画ファイル選択
  - アップロード後、自動的に動画詳細画面へ遷移

### 2. 動画詳細・解析結果確認画面（`/videos/[id]`）

**ファイル**: `frontend/src/app/videos/[id]/page.tsx`

**機能**:
- アップロードした動画の再生
- 動画情報の表示（ファイル名、サイズ、再生時間、アップロード日時、ステータス）
- 解析結果の表示
  - エリクサー分析
  - コスト分析
  - タイミング分析
  - リスク分析
- 解析開始ボタン（未解析の場合）

**レイアウト**:
- 左側: 動画プレーヤー
- 右側: 動画情報と解析開始ボタン
- 下部: 解析結果セクション

### 3. 動画アップロード画面（`/upload`）

**ファイル**: `frontend/src/app/upload/page.tsx`

**機能**:
- 動画ファイルのアップロード
- タイトル入力（任意）

**注意**: この画面は現在、トップ画面のタブで統合されていますが、直接URLアクセス用に残しています。

## 🔄 画面遷移フロー

```
トップ画面 (/)
  ├─ [動画一覧タブ] → 動画カードをクリック → 動画詳細画面 (/videos/[id])
  └─ [アップロードタブ] → アップロード成功 → 動画詳細画面 (/videos/[id])
                              ↓
                      （自動的に解析開始）

動画詳細画面 (/videos/[id])
  └─ [ホームに戻る] → トップ画面 (/)
```

## 🎨 UIコンポーネント

### 共有コンポーネント
- `UploadTab` (`frontend/src/app/components/UploadTab.tsx`)
  - 動画アップロードフォームコンポーネント
  - トップ画面のタブ内で使用

### レイアウト
- `RootLayout` (`frontend/src/app/layout.tsx`)
  - 共通ナビゲーションバー
  - フッター

## 📝 画面設計の原則

1. **シンプルさ**: 3画面のみで完結する構成
2. **タブ切り替え**: トップ画面で主要機能にアクセス可能
3. **動画中心**: すべての画面が動画とその解析結果に焦点
4. **レスポンシブ**: モバイル・タブレット・デスクトップに対応
